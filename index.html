🚀 AWS DevOps CI/CD Pipeline

Cloud Computing with DevOps Internship Project

Project Overview

This project demonstrates the implementation of a Continuous Integration and Continuous Deployment (CI/CD) pipeline using AWS and Jenkins.

The website is automatically deployed on an Amazon EC2 instance whenever changes are pushed to the GitHub repository.

Technologies Used

  • Amazon Web Services (AWS)
  • Amazon EC2
  • Apache Web Server
  • Jenkins
  • Git & GitHub
  • HTML5
  • CSS3
  • Git Bash

CI/CD Workflow

  1. Developer pushes code to GitHub.
  2. Jenkins detects repository changes.
  3. Jenkins clones the repository.
  4. Build process is executed.
  5. Files are deployed to Apache on EC2.
  6. Website is updated automatically.
